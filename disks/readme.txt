INFOCOM GAMES DISK
==================

This disk contains classic Infocom text adventures 
and the VEZZA Z-machine interpreter.

TO PLAY A GAME:
  VEZZA ZORK1.Z3     - Play Zork I
  VEZZA ZORK2.Z3     - Play Zork II
  VEZZA ZORK3.Z3     - Play Zork III
  VEZZA HHIKER.Z3    - Hitchhiker's Guide
  VEZZA PLANETFL.Z3  - Planetfall
  VEZZA ENCHANT.Z3   - Enchanter
  VEZZA WISHBRIN.Z3  - Wishbringer
  VEZZA DEADLINE.Z3  - Deadline
  VEZZA SUSPEND.Z3   - Suspended

VEZZA COMMANDS:
  During game: Type commands like LOOK, GO NORTH
  SAVE     - Save your game
  RESTORE  - Load a saved game
  QUIT     - Exit the game

Games sourced from eblong.com/infocom
VEZZA interpreter from sijnstra.itch.io/vezza
